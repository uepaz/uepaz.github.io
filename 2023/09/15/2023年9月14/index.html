stable diffusion的本地部署手把手教程

Created2023-09-15|Updated2025-04-07
|Word count:444|Reading time:1min|Post View:

教程包括：

1.安装miniconda并利用miniconda单独为sd创建环境

2.安装git

3.安装CUDA

4.在环境中克隆sd代码

5.推荐几个模型下载网站和如何使用模型

注意：如果报错,推荐科学上网

一.安装miniconda

miniconda官网miniconda

安装改一下路径,其它一路next

安装完成后在win搜索里搜minicond打开,输入

conda -V

如果弹出版本号那就没错。

接下来为sd的部署创建环境：

在minicond中输入以下指令,然后回车

conda create —name stable python=3.10.6

它会问你是否安装,输入y即可

没报错就可以

输入并执行下面语句，激活conda虚拟环境并更新pip

conda activate stable

python -m pip install —upgrade pip

pip config set global.index-url https://mirrors.aliyun.com/pypi/simple/

记得按回车

二.安装git

点击git官网

可以改一下安装位置,其它直接next

三.安装CUDA

首先查看自己电脑应该安装哪个版本

在minicond输入

nvidia-smi

看CUDA Version 后面的版本号,在官网下载对应版本

NVIDIA cuda官网

在安装时同样的,可以改变安装位置,其它无脑下一步

接下来克隆sd的文件

现在确定你的minicond界面为

然后输入你想安装的盘

E: 或者其它你想的盘

再输入,克隆sd

git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui.git

然后你就能在你选择的盘里看到文件夹stable-diffusion-webui

在minicond里输入

cd stable-diffusion-webui

再执行下面指令打开应用程序

stable-diffusion-webui

注意,这一步会下载很多依赖,请确保网络通畅,不行的话,请科学上网
